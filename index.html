textContent vs innerText vs innerHTML

As propriedades textContent, innerText e innerHTML são usadas para acessar ou modificar o conteúdo de elementos HTML, mas cada uma se comporta de forma diferente:

  • textContent: Retorna todo o conteúdo de texto, incluindo o que está escondido com CSS. Ex: element.textContent = "Olá <b>mundo</b>" irá mostrar literalmente "Olá <b>mundo</b>".
  • innerText: Retorna apenas o texto visível (respeita o CSS como display: none) e ignora tags HTML. Ex: element.innerText = "Olá <b>mundo</b>" também mostra "Olá mundo" como texto puro, sem aplicar a tag.
  • innerHTML: Interpreta o conteúdo como HTML. Ex: element.innerHTML = "Olá <b>mundo</b>" exibirá "Olá mundo" em negrito.

Use textContent para segurança e performance, innerText quando quiser apenas o texto visível, e innerHTML se for necessário renderizar tags HTML.

Alterando CSS com JavaScript via DOM

Podemos usar JavaScript para alterar estilos CSS de elementos HTML acessando a propriedade style de um elemento. Isso é útil para aplicar estilos dinamicamente sem precisar editar o CSS direto.
